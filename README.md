# Portfolio Website

A clean and simple portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth scrolling navigation
- Clean, modern layout
- Easy to customize

## Customization

1. Replace placeholders in `index.html`:
   - [Your Name]
   - [Your Profession]
   - [Write a short bio...]
   - [your.email@example.com]
   - Add your projects in the projects section

2. Modify `styles.css` to change colors, fonts, etc.

3. Add images:
   - Create an `images` folder
   - Add profile picture, project screenshots, etc.

## How to View

Open `index.html` in your web browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
